7. 与 React 组件的关系

本目录的 standalone SVG 是 React 组件 <VGIMark/>（位于 react/src/trust/VGIMark.tsx）的"无 CSS 双胞胎"。两者：

  • 共用同一 path：盾牌 + 勾选 path 与 PITL 同款（04 §2.3 锁定）
  • 共用同一规格：pill 高度 20/28、内边距 8/12、border 1.5、radius 9999 都来自 ai-tokens.css §3
  • 差异：standalone .svg 把字体作为 font-family 引用，需要消费方加载 Inter / Noto Sans SC；React 组件依赖 ai-tokens.css 已 import 的字体栈
  • 变更同步：spec 变更时两边同步——更新 React 组件后跑 node scripts/build-vgi-svg.mjs 重新生成 .svg
