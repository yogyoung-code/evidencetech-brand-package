4. Diff View · 红绿语义

AI 输出（红色删除线） vs 专家修订（绿色高亮）。颜色复用母品牌 error / success token，跨产品一致。

该患者 必须立即停用阿司匹林 在 INR > 3.0 时建议停用阿司匹林（结合临床判断）。

同时建议 每周复查 INR 在用药调整后 3–5 天复查 INR，之后每 2 周一次。

5. Progressive Disclosure · 三层级

L1 核心结论（产品色 + Bold） → L2 证据摘要（neutral-700） → L3 溯源细节（neutral-500）。层间间距 12px。

建议在合并 PPI 使用时停用阿司匹林。

[1] Smith et al., NEJM 2024; n=2,341; 风险比 0.62 (95% CI 0.48–0.81)。

6. 图像政策

沿用母品牌 v1.1 §8，AI 子品牌不放松任何一条。

允许
真实临床照片
自然光、偏冷中性、医生 / 工作场景，禁止 stock photo。
允许
几何抽象 + dot-spiral 母题
仅产品色 + 中性色；CoT 节点链是 dot-spiral 的演化形式。
禁止
AI 生成医学影像
FTC 合规红线。任何 AI 生成的 CT / MRI / 病理图、人体结构图均不允许。
禁止
"AI 大脑" / 机器人具象
违反"医生在环"语义，给 AI 拟人化错觉。改用几何抽象。
禁止
Stock photo
图库感太强，与"医学严谨"气质冲突。
禁止
Emoji / 卡通 / 贴纸
医学产品禁用情感装饰元素，沿用母品牌 v1.1。
