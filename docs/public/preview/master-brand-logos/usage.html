② Clear-Space 与最小尺寸

定义 X = icon mark 高度（≈ 渲染后 logo 总高 × 2/3）。所有变体四周保留至少 ½ X 净空。Hero / 主视觉位推荐 1 X。

③ A11y 对比度（WCAG 2.1）

评级图例：AAA ≥ 7:1 · AA ≥ 4.5:1 · AA-large ≥ 3:1（图形 / 大字达标，正文不达标） · fail < 3:1。

完整 64 组数据在 a11y-contrast.json。下表展示推荐使用规则中最关键的对比组合。

红线（禁止使用）：
  • default（多色版）放在 brand-de / brand-rcp / navy 等低亮度背景 — 主体 navy 与背景对比 < 3:1
  • reverse 放在浅色背景 — 反白对比直接失败
  • mono-black 放在低明度有色背景 — 检查 ≥ 3:1 才允许
  • 任意变体放在视觉密度大的图片或渐变上（须先加纯色衬底再放 logo）

允许的拆分

  • 允许 Icon mark only — 取 viewBox "371 371 1373 1262" 裁切（favicon / app icon）
  • 允许 EN row only — 直接用 *-en.svg
  • 允许 CN row only — 直接用 *-cn.svg

不允许的拆分

  • 禁止 拆下任意单字（如只显示「梅」字）
  • 禁止 slogan 单独使用（必须随 wordmark）
  • 禁止 改 icon 与 wordmark 的相对位置 / 比例
  • 禁止 改 wordmark 字体（不允许其他字体重排）
  • 禁止 在 default 上替换任意一色（须用 mono / reverse 变体）
  • 禁止 在 icon mark 内部裁切

⑤ PR 校验清单（设计 / 工程师自查）

  • □ Logo 四周保留 ≥ ½ X 净空（X = icon 高，约总高 × 2/3）
  • □ Logo 总高 ≥ 24 px（数字）/ ≥ 6 mm（印刷）
  • □ 小于上述阈值时，已替换为 icon mark only
  • □ 净空区内无其他 logo / 标题 / 装饰元素
  • □ 背景色 / 渐变 与 logo 主体对比 ≥ 3:1（图形阈值）
  • □ 选中的变体匹配背景：白底 → default 或 mono-navy；深底 → reverse；浅灰底 → mono-navy / mono-black
  • □ Co-brand 场景下分隔线高 = 母品牌 logo 高 × 0.6，颜色 #CBD5E1（反白态用 30% 白）
  • □ 印刷场景下，logo 宽度 < 25 mm 时已切换为 mono 单色版
  • □ default 多色版未被任意修改（不增减颜色 / 不调色相）
  • □ wordmark 未被任意改字体或字间距
