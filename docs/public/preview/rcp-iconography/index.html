6. 决策对比

A · 硬角精修 r=0

优势：与战略 PDF 中"锐脉"语义最贴合（pulse = 锐利信号）。在 16/20px 小尺寸下对比度最强。
劣势：与 DE 并排时几何风格不同源（DE 有 corner arc，RCP 全直线），家族 rhyme 弱。
路径长度：~50（最短，stroke-draw 动画最快）

B · 软角 fillet r=0.5/1.0

优势：保留 ECG 锐度的同时削掉硬角"刺"。中间路线，对小尺寸打印友好。
劣势：与 DE 配方不一致——既不"完全锐利"也不"完全家族化"，可能两边都不到位。
路径长度：~52（中等）

C · 与 DE 几何对齐 推荐

优势：与 DE icon 共用 fillet 配方（peak r=0.25 / baseline r=1.5）。两个 ECG 图标在矩阵 lockup（02 §4.1）中并排时家族 rhyme 最强。
劣势：sub-pixel 在 16px 下 corner arc 可能渲染为 1px 抗锯齿模糊（仍清晰，但比 A 略软）。
路径长度：~53（最长，但 stroke-draw 动画感觉一致）

7. 决策

选 A：保留极简硬角；选 B：折中；选 C：与 DE 同源（推荐）。
选定后将更新 react/src/icons/RCPPulseIcon.tsx，并产出 preview/rcp-iconography/icon.svg、icon-reverse.svg、icon-animated.svg。
