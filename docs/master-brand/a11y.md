## 推荐使用规则（基于矩阵）

| 背景 | 推荐变体 | 原因 |
|---|---|---|
| #FFFFFF / #F8FAFC（白/纸白） | default 或 mono-navy | navy 在白底 ≥ 13:1（AAA）；多色版 cyan 在白底约 2.5:1，作为 icon 高光不构成 a11y 风险（非主体） |
| #E2E8F0（浅灰） | mono-navy 或 mono-black | default 在浅灰底视觉对比降低；mono 单色版更稳 |
| #0B1220 深底 | reverse | 反白专用版本 |
| 产品色背景（蓝/紫） | reverse 或 mono-black（视亮度） | 深产品色用 reverse；浅产品色 < 3:1 时禁止放 logo |
| 渐变 / 图片背景 | 禁止 | 需先加纯色色板再放 logo |

## 红线

- ❌ default（多色版）放在低亮度产品色背景（如 brand-de、brand-rcp、navy）—— 主体 navy 与背景对比 < 3:1
- ❌ reverse 放在浅色背景 —— 反白对比直接失败
- ❌ mono-black 放在 brand-se（紫）以外的所有暖紫/低明度背景 —— contrast < 3:1
- ❌ 任意变体放在视觉密度大的图片或渐变上（无纯色衬底）
